About this site

You might ask why this site is plain HTML/CSS, instead of a frontend framework like Vue or React. I've rewritten this site several times, using React, Google Sites, Wordpress, and ASP.NET.

In the end, it's just a case of always using the right tool for the job, and I really don't think that the other tools and framework made the better in any substantial way.

A static site using 'pure' HTML is easy to maintain, cheap to host, and super fast. The tools I used are
  • Visual Studio Code
  • GitHub
  • NPM
  • Web Accessability Evaluation Tool
